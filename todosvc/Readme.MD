# todosvc 
* 할일 목록 서비스
---------------------------------------
## 실행 방법
* 의존성 설치
~~~
npm install
~~~
* 개발 모드로 실행
~~~
npm run start:dev
~~~
* 운영 모드로 실행
~~~
npm run start
~~~
----------------------------------------

## 사용방법 확인
* http://localhost:8000 에서 오른쪽 사이드 메뉴 확인
